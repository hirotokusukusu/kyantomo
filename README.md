# Kyantomo（キャン友）

大学生のためのコミュニティアプリ

---

## 🎯 コンセプト

「大学生活に特化したSNS」

同じ学部・同じ授業・同じ趣味の学生とつながれる
学内限定のコミュニティアプリです。

大学内の情報の非対称性を減らし、
“つながりにくさ”を解消することを目的に開発しました。

---

## 🚀 開発背景

・友達作りに不安を感じる学生が多い  
・授業情報が口コミベースで閉じている    

大学内で口頭アンケートを実施したところ、
66％の学生が「このようなアプリがあれば使いたい」と回答。

この課題を解決するために開発を開始しました。

---

## 💡 主な機能

- 大学ドメイン（@kwansei.ac.jp）限定登録
- プロフィール作成（学部・学年・趣味・履修授業）
- 掲示板投稿機能
- いいね・返信機能
- 友達申請・承認
- 1対1メッセージ機能

---

## 🛠 技術構成

- React Native（Expo Router）
- TypeScript
- TanStack Query
- Supabase（Auth / PostgreSQL / RLS）
